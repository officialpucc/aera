# Aera Product Backlog

## Epics & Stories

### Epic 1: The Central Cortex
*   **Story 1.1:** Contextual Awareness - Ingest all real-time financial and operational data.
*   **Story 1.2:** Long-term Synaptic Memory - Hybrid vector-relational memory system.
*   **Story 1.3:** The Neural Bottleneck - Single point of contact for all departments.

### Epic 2: Sensory Dendrites
*   **Story 2.1:** Supply Chain Dendrite - Real-time inventory from Portes.
*   **Story 2.2:** Financial Dendrite - Ingest COGS, burn rates, and bank balances.
*   **Story 2.3:** Sales/Rental Dendrite - Track customer interactions and predict demand.

### Epic 3: Predictive Motor Output
*   **Story 3.1:** Predictive Restock - Predict Time-to-Zero and propose orders.
*   **Story 3.2:** Autonomous Negotiation Simulation - Simulate vendor pricing scenarios.

## Sprint 1: The MVP Brainstem
*   **A-001:** Implement Aera Core (Node.js/TS) with singular context management.
*   **A-002:** Create the "Perforated" Data Ingestion Interface (The Dendrite SDK).
*   **A-003:** Develop Supply Chain Dendrite (Porte Simulation + Inventory Logic).
*   **A-004:** Implement "Action Proposal" notification system (The User Loop).
*   **A-005:** Build Financial Ledger Dendrite (Basic COGS/Revenue tracking).
